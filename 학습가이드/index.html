LEARNING GUIDE

초·중·고 학습관리 가이드: 진단부터 오답 재학습까지

공부 시간을 늘리기 전에 어디에서 막혔는지, 계획이 실제로 실행되는지, 틀린 문제가 다시 학습으로 이어지는지를 확인해야 합니다. 아래 순서에 따라 학생에게 필요한 관리 기준을 살펴보세요.

GUIDE 01

초기 상담에서는 반 배정보다 우선순위를 정합니다.

현재 점수만 보면 아이의 진짜 어려움이 잘 보이지 않습니다. 같은 점수라도 개념이 부족한 학생, 연습량이 부족한 학생, 시험 시간 배분이 어려운 학생은 전혀 다른 관리가 필요합니다.

📘

현재 교재

어떤 교재를 어디까지 진행했는지 확인해 시작 지점을 잡습니다.

🧭

학습 습관

숙제, 복습, 시험 준비 시점, 집중 흐름을 함께 살펴봅니다.

🎯

목표 정리

성적 향상, 결손 보완, 내신 대비, 습관 개선 중 우선순위를 정합니다.

GUIDE 02

진단은 학생을 평가하는 절차가 아니라, 출발점을 찾는 과정입니다.

수학은 개념, 계산, 조건 해석, 응용 접근을 나누어 보고 영어는 어휘, 문법, 문장 구조, 독해 속도, 본문 복습을 나누어 확인합니다.

A

교과 진단

어느 단원부터 막혀 있는지 확인합니다.

B

행동 진단

계획과 실제 실행 사이의 간격을 확인합니다.

C

오답 진단

틀린 문제의 원인을 유형별로 분류합니다.

D

계획 조정

교재, 숙제량, 복습 주기, 시험 대비 일정을 조정합니다.

GUIDE 03 · BY GRADE

초등·중등·고등은 같은 방식으로 관리하지 않습니다.

학년이 올라갈수록 과목 수와 시험 범위가 늘고 스스로 조정해야 할 일이 많아집니다. 현재 학년에 맞는 핵심 기준부터 확인합니다.

초등

시작 시간과 기초 확인

어휘·독해·연산처럼 다음 학습의 바탕이 되는 기초와 정해진 시간에 공부를 시작하는 습관을 함께 봅니다.

중등

학교 진도와 수행 연결

개념 복습, 문제 적용, 서술형, 수행평가를 시험 일정에 맞춰 배치하고 미완료 과제를 다음 계획에 반영합니다.

고등

취약 과목과 시간 배분

모든 과목을 같은 비중으로 잡기보다 현재 성취도와 시험 일정을 기준으로 보완 순서와 주간 학습량을 정합니다.

GUIDE 04

플래너는 ‘오늘 공부하기’가 아니라 완료 기준을 적는 도구입니다.

좋은 플래너는 과목, 단원, 분량, 완료 기준이 분명합니다. 그리고 실행 결과를 확인해 다음 계획으로 이어집니다.

목표

구체적으로 적기

“수학 하기”보다 “일차함수 유형 1~20번 풀이”처럼 끝이 보이게 적습니다.

실행

결과를 표시하기

완료 여부, 걸린 시간, 어려웠던 부분, 틀린 개수를 함께 봅니다.

수정

다음 계획에 반영하기

못 끝낸 이유가 양의 문제인지 습관 문제인지 확인해 조정합니다.

GUIDE 05 · WRONG ANSWERS

오답은 정답을 베끼지 않고, 틀린 원인을 분리해 다시 학습합니다.

같은 문제를 다시 푸는 것만으로는 반복 실수를 줄이기 어렵습니다. 원인을 구분한 뒤 필요한 개념과 유사 문제로 돌아갑니다.

개념 부족

개념과 예제로 돌아가기

공식이나 문법 규칙을 다시 확인하고 기본 예제를 스스로 설명할 수 있는지 봅니다.

조건 해석

문제의 요구 표시하기

주어진 조건과 구해야 하는 것을 분리해 읽고 풀이 전에 필요한 정보를 표시합니다.

계산·표기 실수

실수 발생 지점 기록하기

부호, 단위, 철자처럼 반복되는 실수를 유형으로 남겨 다음 풀이의 확인 항목으로 사용합니다.

시간 부족

문제별 시간 기준 세우기

오래 붙잡은 문제와 건너뛸 문제를 구분하고 제한 시간 안에서 풀이 순서를 다시 연습합니다.

GUIDE 06 · EXAM PLAN

시험 준비는 범위를 나눈 뒤 확인과 재학습 시간을 남겨둡니다.

아래 흐름은 일반적인 예시입니다. 실제 기간과 학습량은 학교 일정과 학생의 현재 수준에 맞춰 조정합니다.

4주 전

개념 복습과 기본 문제 풀이로 단원별 이해 정도를 확인합니다.

2주 전

새로운 분량보다 오답 재풀이와 암기 확인, 시간 배분에 집중합니다.

GUIDE 07 · FEEDBACK

학부모 피드백은 진도보다 다음 관리 기준이 보여야 합니다.

진행

과목과 단원, 사용 교재, 완료한 분량을 구체적으로 확인합니다.

GUIDE FAQ CHECK

학습가이드를 볼 때 자주 묻는 질문

상담 전 학부모님이 가장 많이 궁금해하는 진단, 플래너, 오답 관리 기준을 짧게 정리했습니다.

학습 진단은 한 번만 하면 되나요?

처음에는 출발점을 찾고, 이후에는 수업·숙제·오답 기록을 보며 계속 조정합니다. 학생의 수행 흐름이 바뀌면 계획도 함께 바뀌어야 합니다.

플래너는 왜 필요한가요?

무엇을 얼마나 했는지, 왜 못 끝냈는지, 다음에는 어디를 줄이거나 늘려야 하는지 확인하기 위한 실행 기록입니다.

오답 관리는 어떻게 이어지나요?

틀린 문제를 다시 푸는 데서 멈추지 않고 개념 부족, 계산 실수, 조건 해석, 시간 부족처럼 원인을 나눠 재학습합니다.

오답은 틀린 기록이 아니라, 다음에 맞힐 수 있게 만드는 지도입니다.

개념 부족, 계산 실수, 조건 누락, 시간 부족처럼 원인을 분리하면 같은 문제를 반복해서 틀리는 일을 줄일 수 있습니다.
